# Flask Microservice

Production-ready Flask 3 microservice with Celery task queue, Redis, health checks, Prometheus metrics endpoint, and Docker Compose setup.

**Stack:** Flask | **Complexity:** complex
**Tags:** `flask`, `microservice`, `celery`, `redis`, `async`, `python`, `docker`

## Quick Start

```bash
pip install -r requirements.txt
flask run --reload
```

## Docker

```bash
docker compose up --build
```
